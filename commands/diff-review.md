# Diff Review Command

このコマンドは、直近の git diff を対象として  
**diff-review サブエージェント**を起動します。

用途：
- 実装直後
- コミット前
- PR 前の差分整理

---

## 実行内容

1. git diff を取得する
2. 変更されたファイルを特定する
3. diff-review サブエージェントに処理を移譲する

---

## サブエージェント呼び出し

以下のサブエージェントを使用してください。

- **name**: diff-review

サブエージェントは以下を担当します：
- 差分レビュー
- 安全な自動修正
- 複数サイクルでの品質改善
- レポート出力
